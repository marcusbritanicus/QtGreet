QtGreet - A light-weight Qt-based Greeter

QtGreet is a cross-desktop graphical display manager for linux built using Qt5. It supports two backends, namely greetd and lightdm .

Some of its features are

  • Cross-desktop - works on X11 and Wayland
  • Light-weight - Pure Qt5 + lightdm + greetd
  • Standards compliant - PAM, logind support
  • Supports remote login - (via lightdm)
  • Supports guest sessions - (via lightdm)
